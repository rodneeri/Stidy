# Possible Features to add in the Long Term

The ambitious horizon for [[STiDY]] — bigger bets, deferred from v1 for stability/scope. Not committed;
seeds for when the foundation is solid.

## Intelligence
- **Knowledge graph** — `pgvector` is already enabled. Embed resources/notes → a navigable concept
  graph linking subjects, topics, and materials (very on-brand with this Obsidian vault metaphor).
- **AI tutor / active recall agent** — adaptive questioning that targets weak spots from grade + SRS data.
- **Lecture audio summarizer** — upload a recording → transcript → structured notes + flashcards.
- **Citation generator** — pull references from resources, format in APA/MLA/etc.
- **Burnout analytics** — surface trends from `study_logs` + task load; gentle interventions.

## Reach & platform
- **Multi-country admission engine** — generalize nota de corte into a worldwide cutoff/requirements
  engine (the `goals` table is designed for this). Oposiciones, US/UK admissions, etc.
- **Realtime collaboration** — co-work via the unique `handle` (study groups, shared decks/timetables).
- **Mobile / PWA** — installable, offline-capable focus + flashcards.
- **Study-block auto-injection** — a calendar engine that schedules study sessions around exams
  (deferred once already as too risky; the AI chat answers "should I go to class?" for now).
- **Syllabus marketplace** — share/import parsed grading structures between students.

## Annotation
- **In-app PDF/video annotation** — highlight + note over stored resources (the viewer exists; this is
  the next layer).
